# scrapper_comparison

## Setup Instructions

### Prerequisites
- Python 3.8 or higher
- `pip` (Python package manager)

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd vendorVisualizations
   ```
3. Install the required Python packages:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Analysis
1. Open the Jupyter Notebook:
   ```bash
   jupyter notebook file_comparison_analysis.ipynb
   ```
2. Follow the instructions in the notebook to load and analyze the data files.

### Data Files
- Place your data files in the respective folders (`target`, `wayfair`, `zarahome`) before running the analysis.
